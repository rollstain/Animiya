# design/ — выгрузка из claude.ai/design

Складывай сюда результаты работы дизайн-инструмента, потом скажи мне «глянь design/».

## Что и в каком виде класть (по убыванию полезности для меня)

### 1. Скриншоты экранов — PNG/JPG (лучший вариант)
Я читаю изображения визуально. Экспортни/заскриншоть каждый экран и положи сюда:
```
design/screens/home.png
design/screens/title-card.png
design/screens/dub-sheet.png
design/screens/player.png
design/screens/library.png
...
```
Именуй по экрану — я пойму, что где.

### 2. Экспортированный код (React/HTML/CSS) или токены
Если инструмент отдаёт код или спецификацию токенов — сохрани файлами:
```
design/tokens.json     (или .css — цвета, отступы, радиусы, типографика)
design/components/...   (сгенерированный код компонентов)
```
Я прочитаю и переведу в Compose.

### 3. Спецификация словами
Любой текст/markdown с описанием (палитра, размеры, поведение) — тоже сюда.

## Чего НЕ нужно
- Не удаляй ничего из корня проекта, клади только в `design/`.
- Не переживай про формат — что выгрузилось, то и клади, разберусь.
