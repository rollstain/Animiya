# Animiya — токены дизайн-системы (тёмная тема)

Постер-центричная система для аниме-стриминга. Тёмная база, один фиолетовый акцент.
Elevation передаётся **осветлением** surface-контейнеров, не тенями.

## Шрифты
- **Unbounded** — крупные заголовки, лого (weights 400–800)
- **Onest** — весь UI-текст (weights 400–700)
- Обе с кириллицей, лицензия OFL 1.1.

## Elevation (осветление, не тени)
Формула overlay: `α = min(1, (4.5·ln(1+dp)+2)/100)`
Уровни 0/1/3/6/8/12dp → 0 / 5.1 / 8.2 / 10.8 / 11.9 / 13.5 %
Практическая карта: 0 `#141218` · 1 `#1D1B20` · 3 `#211F26` · 6 `#2B2930` · 12 `#36343B`

## Типошкала (M3, sp)
- Display: 57 / 45 / 36

- Title: 22 / 16 / 14

- Label: 14 / 12 / 11

Заголовки — Unbounded (600–800). Title/Body/Label — Onest.

## Отступы (4dp-грид)
`4 · 8 · 12 · 16 · 24 · 32 · 40 · 48`
Боковое поле экрана: **16**. Промежуток сетки постеров: **12**.

## Радиусы (ShapeTokens)
`0 · 4 · 8 · 12 · 16 · 20 · 28 · 32 · full`
- Постер / карточка: **12**
- Bottom sheet: **24** (только верхние углы)
- Чип: **8**
- Кнопка-пилюля / статус: **full**

## Постер
Соотношение **2:3**, `ContentScale.Crop` (единого 2:3 у источников нет — crop как нейтральный дефолт).
Динамический цвет фона: AniList `color` где есть, иначе MaterialKolor (KMP). НЕ `androidx.palette` (Android-only).

## Motion
- Short 50–200 мс · Medium 250–400 мс · Long 450–600 мс
- Easing Standard `(0.2, 0, 0, 1)` · Emphasized вход `(0.2, 0, 0, 1)`

## Градиент-скрим (текст поверх постера)
Линейный к фону, чёрный `0 → 0.7–0.85 alpha` в нижней трети.
Пример: `linear-gradient(to top, #141218 4%, rgba(20,18,24,0.5) 42%, rgba(20,18,24,0) 72%)`

## Примечание по фреймворку
Не наследоваться от `MaterialExpressiveTheme`. Брать числа (формы/motion/типографику)
в свой слой токенов поверх обычного `MaterialTheme`.
